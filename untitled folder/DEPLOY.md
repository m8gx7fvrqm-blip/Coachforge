# How to Deploy Your CoachForge Website

## Quick Deploy Options

### Option 1: Netlify Drop (Easiest - 2 minutes)
1. Visit: https://app.netlify.com/drop
2. Drag and drop this entire folder onto the page
3. Get instant live link!

### Option 2: GitHub Pages (Free & Permanent)
1. Create a GitHub account (if you don't have one)
2. Create a new repository
3. Upload these files:
   - index.html
   - styles.css
   - script.js
4. Go to Settings > Pages
5. Select main branch
6. Your site will be live at: `https://yourusername.github.io/repository-name`

### Option 3: Vercel (Also Easy)
1. Visit: https://vercel.com
2. Sign up (free)
3. Click "Add New Project"
4. Drag and drop this folder
5. Get instant deployment!

## Files to Deploy
- index.html

- script.js

All files are ready to go - no build process needed!
